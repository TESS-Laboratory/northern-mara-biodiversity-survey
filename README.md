# northern-mara-biodiversity-survey
Created as part of Milcah Kirinyet's Oppenheimer Impact Scholar project, working with Dascot ltd., Sustain East Africa, and Enonkishu Conservancy
This repository contains R scripts and data
The folder **data** contains avian and vegetation data collected during the survey 
The folder **images** contains some of the images taken during the field exercise, the transect photos and some of the bird species captured during the survey
